# TaskTail

TaskTail - веб-приложение для создания и отслеживания личных планов.

В проекте есть личный кабинет, планы по датам, загрузка фото, мотивационные страницы и интерактивная snake-механика.

## Что есть в проекте

- регистрация и вход пользователей
- профиль пользователя
- добавление и отслеживание планов
- отметка выполненных планов по датам
- загрузка фото к планам
- страница мотивационных новостей
- страница аффирмаций
- форма обратной связи
- интерактивная snake-механика

## Технологии

- React
- Vite
- React Router
- Express
- Node.js
- CSS

## Структура

- `client` - клиентская часть на React
- `server` - серверная часть на Express

## Запуск

Сначала нужно установить зависимости для сервера:

cd server

npm install

npm start

Сервер запускается на:

http://localhost:3000

После этого во втором терминале запускается клиент:

cd client

npm install

npm run dev

Клиент обычно открывается на:

http://localhost:5173

## Данные пользователей

Файл `server/users.json` сейчас пустой:

`[]`

После регистрации новые пользователи и их планы сохраняются туда локально.
